# Avanti Image Processor - Releases

Public releases repository for [Avanti Image Processor](https://github.com/JFlashdata/aipImageProcessor).

## Download

Visit the [Releases page](https://github.com/JFlashdata/aipImageProcessor-releases/releases) to download the latest version.

## About

This repository contains only compiled releases of the application. The source code is maintained in a private repository.

## Installation

1. Download the latest `AvantiImageProcessor.exe` from the releases page
2. Run the executable
3. The application will automatically check for updates

## System Requirements

- Windows 10 or later
- No additional dependencies required (standalone executable)
